# PRIVACY POLICY

**Last updated: 12/06/2026 12:54 PM**

This Privacy Policy explains how the Discord bot processes information. By using the Bot, you agree to this Policy.

---

## 1. Information Processed

The Bot may access, process, and store information that is made available through the Discord API, Discord gateway, and discord.py library, strictly as required for functionality and within the limits of permissions and intents granted by Discord.

This may include, depending on enabled features:

- Discord user IDs  
- Discord server (guild) IDs  
- Channel and thread IDs  
- Message content (where required for command or feature functionality and permitted by Discord)  
- Message metadata (including timestamps, message IDs, edits, deletions, and attachments)  
- User profile information provided by Discord (such as usernames, display names, and avatars)  
- Server configuration data (such as roles, permissions, and settings)  
- Interaction data (such as slash commands, buttons, and select menus)  
- Event data provided through the Discord gateway (such as message events, member joins/leaves, and reactions)  

No data is accessed outside what is allowed under Discord’s Terms of Service.

---

## 2. Purpose of Processing

Any processed data is used solely for:

- providing Bot functionality  
- handling commands and interactions  
- server-specific configuration and features  
- debugging, logging, and performance monitoring  
- abuse prevention and system integrity  

---

## 3. Data Storage

Data may be:

- temporarily processed in memory  
- stored in a database or file system if required for functionality  
- logged for debugging or operational purposes  

Data retention is not guaranteed and may change depending on feature requirements.

---

## 4. Data Sharing

The Bot does not sell or trade personal data.

Data may only be shared where necessary for:

- Discord API functionality  
- hosting or infrastructure services  
- legal or regulatory compliance  

---

## 5. Security

Reasonable measures are taken to protect data, but no system is completely secure. Absolute security cannot be guaranteed.

---

## 6. Changes to Processing

The Bot may be updated at any time. New features may require processing additional data made available through Discord’s API and gateway events, but only to the extent required for functionality.

---

## 7. User Rights

Where applicable, users may request access to or deletion of their data, subject to technical and operational limitations.
